# vuepress-plugin-musicplayer

## How to use

1. install `npm i vuepress-plugin-musicplayer -S`

2. add in `config.js`

```js
module.exports = {
  plugins: [
    "vuepress-plugin-musicplayer",
  ]
}
```

## Tip

- This plugin supports `vuepress 1.x`
- This plugin will randomly change the white cat or black cat when router path change.
