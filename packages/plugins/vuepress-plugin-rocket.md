# vuepress-plugin-rocket
back-to-top rocket in vuepress
### Install
```
npm install vuepress-plugin-rocket -S
```

### Use
add in `.vuepress/config.js`
```
plugins: [
  'vuepress-plugin-rocket'
]
```
