# Vuepress Shiki

Fork from https://github.com/octref/shiki

和原版完全相同，但是作者合入太慢了，弄个私有版本先用着。

```bash
yarn global add vuepress@next 
yarn add -D @vutex/vuepress-plugin-shiki
```

in `.vuepress/config.js`:

```js
module.exports = {
    plugins: [
        [
            '@vutex/vuepress-plugin-shiki',
            {theme: 'monokai'}
        ],
    ],
}
```
