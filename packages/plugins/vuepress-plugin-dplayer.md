## Install

```npm
yarn add -D vuepress-plugin-dplayer
# OR npm install -D vuepress-plugin-dplayer
```

## Use

1). `config.js`

```javascript
module.exports = {
  plugins: ['dplayer']
}
```

2). `*.md`, [click here](https://dplayer.js.org/) for more configuration of dplayer.

```markdown
<d-player 
    :options="{
        ...
    }"
/>
```
