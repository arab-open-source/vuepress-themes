# vuepress-theme-demo

基于vuepress的带主题的脚手架



## Experience

```bash
# create
npx emjio-cli init my-blog
# oa
npm install  -g
emjio-cli  init my-blog

# install
cd my-blog
npm install

# run
npm run dev

# build
npm run build
```

**if yarn**

```bash
# create
npx vuepress-theme-emjio-cli  init my-blog
#or
yarn global add vuepress-theme-emjio-cli 
emjio-cli  init my-blog

# install
cd my-blog
yarn install

# run
yarn dev

# build
yarn build
```
