## Install

```bash
$ npm install vuepress-plugin-social-share -D
# OR
$ yarn add vuepress-plugin-social-share -D
```

## Usage

Add config to `.vuepress/config.js`:

```js
module.exports = {
  plugins: ['social-share'],
}
```

:mag: [Custom options detail](https://sns.goyfe.com/guide)
