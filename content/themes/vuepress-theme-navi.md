## 开发主题

> 基于 vuepress-theme-reco 二度开发集成新特性  
> 如果你要共同维护这个主题，请彻底熟悉 vuepress 以及 vuepress 的主题开发协议规范  
> [vuepress 主题开发协议文档](https://vuepress.vuejs.org/zh/theme/)  

## Use

在一个 `vuepress` 项目中使用

将本主题作为依赖安装到项目之中

```bash
npm i vuepress-theme-navi
```

> /docs/.vuepress/config.js

```js
module.exports = {
	// ...
	theme: 'navi',
}
```
